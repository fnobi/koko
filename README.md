koko
====

instant file server (proxy server).

Koko start server on port picked randomly,
and current directory is set as document root.

### Install
```bash
% npm install -g koko
```

### usage
```bash
% cd your/document/root
% koko
document root : your/document/root
listen port   : 59733
```

or, with path

```bash
% koko your/document/root
document root : your/document/root
listen port   : 1835
```
